SETUP & DEPLOYMENT INSTRUCTIONS
================================================================================

DEVELOPMENT SETUP:
  1. cd /sessions/great-ecstatic-heisenberg/scouter
  2. npm install
  3. echo "GEMINI_API_KEY=your_key" > .env.local
  4. npm run dev
  5. Open http://localhost:3000

PRODUCTION BUILD:
  npm run build
  npm start

VERCEL DEPLOYMENT:
  vercel
  (Configure GEMINI_API_KEY in Vercel dashboard)

DOCKER DEPLOYMENT:
  Use Node.js 18+ official image
  Run: npm install && npm run build && npm start

================================================================================
API ENDPOINTS
================================================================================

POST /api/analyze
  Purpose: Analyze website using Google PageSpeed Insights
  Input:   { "url": "https://example.com" }
  Output:  { "url": "...", "scores": [...], "issues": [...] }
  Time:    10-30 seconds (depends on Google's rate limiting)

POST /api/propose
  Purpose: Generate sales proposal using Google Gemini API
  Input:   { "url": "...", "scores": [...], "issues": [...] }
  Output:  { "url": "...", "proposal": "..." }
  Time:    5-15 seconds (depends on Gemini API response)
