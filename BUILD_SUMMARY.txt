QUICK START
===============================================================================

1. Navigate to project:
   cd /sessions/great-ecstatic-heisenberg/scouter

2. Install dependencies:
   npm install

3. Set up environment:
   echo "GEMINI_API_KEY=your_key_here" > .env.local
   
   Get API key from: https://aistudio.google.com

4. Run development server:
   npm run dev

5. Open browser:
   http://localhost:3000

6. Test the app:
   - Enter a website URL (e.g., example.com)
   - Click "スカウト開始"
   - Wait for analysis (10-30 seconds)
   - View radar chart and weakness report
   - Proposal generates in 5-15 seconds
   - Click copy button to copy proposal

   - HTTPS recommended for production
